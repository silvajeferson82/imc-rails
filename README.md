# API IMC

## Requisitos Necessários:

* Ruby 3.0.0
* gem 'rails', '~> 6.1.4'



## Para executar o projeto:

### Clone em sua máquina

```shell
git clone https://github.com/silvajeferson82/imc-rails.git
```

### Instale as dependências
```shell
bundle install
```


### Execute a aplicação
```shell
rails s
```
